Classes

Classes are a package of methods and variables that can be reused. It helps prevent really complcated variable names. You can assign proerties to the class like hair color, eye color and job. I will make an example class and use commits to explain it.


#declare class person
class person

#During the initializes it takes name, age, job. Job as a default unemployed if you dont put a job in.

#This sets the instance variables. This is info that can be used in all the class.
@name = name

@job = job

end

#when you call this you can update the job.
