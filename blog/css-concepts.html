Display: Inline vs Inline block



When you use display:inline you can adjust the margins and padding of the elements left and right. However you cannot set height or width. Inline will allow other elements to sit to the left and right of each other.

Display:Inline block works like inline but you can set all margins, padding, height and width.

These can be used with any type of element. The real trick is knowing which one to use when.
