Arrays and Hashes



Arrays is a variable that can store more then one variable. It will create an index of the variables starting at zero with each of the new variables. It can index nil as a variable. You can assign each index with a number or it can assign them.

Hash is like array however where arrays store a value using a integers as an index a hashs store a value at an index of your making. So it is more like storing a pair of data.

You can think of a hash as an Array that you can modify the index.
