Enumerable map

Enumerable map is a great way to go through arrays or hashs. It has amazing short hand ways to iterate through them. With this short hand stuff you have to balance how easy the code is to read and how short it is. You do not want to scaurfice readability just to make the code look cool.

You can use this tool to help debug some code. It is a quick way to iterate through an array. Lets say you have an array and they all should be true. You can go into the code and add lambda syntax to check for true. "p array.map(&:odd?)". This will print out everything in the array for you to check. A lot faster then setting up a for each loop.

So here are the basic set up for non lambda. arry.map { |num| num.to_i} | 1 | 2 |3| 4 | 5 | 6 | 7| 1: Calling the object. In this case an array. 2: Calling the .map on the array. 3: Open the { this starts what is like a for each loop. 4: Here is where we take assign each iteration a name. In this case we are calling it num. 5: This is where you start doing things to the iteration of the array. These actions can be basic or advance. 6: action taken on the iteration. 7: Ends the "for each loop"

Thank you for reading about map and Enumerable.
